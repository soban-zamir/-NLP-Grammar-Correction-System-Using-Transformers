• This project was completed in collaboration with Mr. Muhammad Umer (Computer Engineering, Class of 2028, Habib University, Karachi).

• It focuses on developing an interactive grammar correction system using Machine Learning techniques involving Python and Gradio. The application provides real time grammatical feedback on user entered text through a simple and user friendly web interface.

• Uses Python, transformers (like BERT/T5), PyTorch and Gradio to provide real time grammar corrections on Colab.
